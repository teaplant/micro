# 用户协议 GitHub Pages

这是一个可以直接部署到 GitHub Pages 的静态用户协议页面。

## 部署方式

1. 在 GitHub 创建一个公开仓库，例如 `user-agreement`。
2. 把本目录内的文件推送到仓库的 `main` 分支。
3. 进入仓库的 `Settings` -> `Pages`。
4. Source 选择 `Deploy from a branch`。
5. Branch 选择 `main`，Folder 选择 `/root`，保存。

部署完成后，页面地址通常是：

```text
https://你的用户名.github.io/仓库名/
```

记得把 `index.html` 里的联系邮箱替换为你自己的邮箱。
